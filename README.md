PROJECT SUMMARY

This project analyzes historical sales data from Bright Coffee Shop to provide actionable insights for the new CEO. Using SQL, Databricks, and visualization tools, the analysis identifies top revenue-generating products, peak sales times, and overall performance trends. Key recommendations include targeted marketing during slow periods, stocking more of best-selling items, and promoting underperforming products. The deliverables—data pipeline plan, processed dataset, dashboards, and presentation—equip leadership with clear strategies to grow revenue, optimize product performance, and enhance customer engagement.
